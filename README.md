# DOT Ongkir

Hiring task DOT Indonesia.

## Development Guide

Engines:

- `Nodejs: V20.10.0`

Steps:

- Clone

`git clone git@github.com:nuzulh/dot-ongkir.git && cd dot-ongkir`

- Install pnpm

`npm i -g pnpm`

- Install packages

`pnpm install`

- Start development server

`pnpm dev`

- Build to static files

`pnpm build`

Done.
